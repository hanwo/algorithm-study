# 문제 설명

정수 배열 `nums`가 주어졌을 때, 좋은 쌍의 개수를 반환하세요.

쌍 `(i, j)`가 좋은 이유는 `nums[i] == nums[j]`이고 `i < j`일 때입니다.

## 예제 1

- 입력: `nums = [1, 2, 3, 1, 1, 3]`
- 출력: `4`
- 설명: 다음과 같은 4개의 좋은 쌍이 있습니다. `(0,3)`, `(0,4)`, `(3,4)`, `(2,5)` (0-인덱스 기준)

## 예제 2

- 입력: `nums = [1, 1, 1, 1]`
- 출력: `6`
- 설명: 배열의 모든 쌍이 좋습니다.

## 예제 3

- 입력: `nums = [1, 2, 3]`
- 출력: `0`

## 제한사항

- `1 <= nums.length <= 100`
- `1 <= nums[i] <= 100`
